Chapter 3 Function Reference

3.1 code_responses()

code_responses(coding_scheme, responses, manual = "null")

Codes one response object with one coding scheme.

Arguments:

  • coding_scheme: list or character string containing a coding scheme from IQB Studio Lite.
  • responses: list or character string containing unit responses from IQB Testcenter.
  • manual: optional list or character string with manual codes. The default "null" is treated like missing manual data.

Returns a data frame.

3.2 code_responses_array()

code_responses_array(
  coding_scheme,
  responses,
  collapse = ";;;",
  wrap_start = "[[[",
  wrap_end = "]]]",
  missing = "___MISSING___"
)

Codes several response objects with one coding scheme.

Arguments:

  • coding_scheme: list or character string containing a coding scheme from IQB Studio Lite.
  • responses: list or character string containing an array of response objects.
  • collapse: separator for concatenating list values.
  • wrap_start: prefix used when wrapping list values.
  • wrap_end: suffix used when wrapping list values.
  • missing: placeholder that is converted to NA.

Returns a data frame. The value column is coerced to character, so list entries can be represented in a rectangular table.

3.3 get_dependency_tree()

get_dependency_tree(coding_scheme)

Displays the variable dependency tree of a coding scheme.

Arguments:

  • coding_scheme: list or character string containing a coding scheme from IQB Studio Lite.

Returns a data frame with dependency information.

3.4 extract_variable_location()

extract_variable_location(
  units,
  collapse = ";;;",
  wrap_start = "[[[",
  wrap_end = "]]]",
  missing = "___MISSING___"
)

Extracts variable locations from unit definitions.

Arguments:

  • units: tibble with a definition column containing unit definitions.
  • collapse: separator for concatenating list values.
  • wrap_start: prefix used when wrapping list values.
  • wrap_end: suffix used when wrapping list values.
  • missing: placeholder that is converted to NA.

Returns a data frame.
